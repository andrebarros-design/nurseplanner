# Nurse Planner - HPA Planeamento
A modern, responsive roster management application for HPA nurses.

## Features
- **Monthly & Daily Views**: Easily switch between high-level monthly overviews and detailed daily schedules.
- **Shift Management**: Supports various shifts (Morning, Interim, Afternoon, Night) with visual indicators.
- **Personalized Dashboard**: Filter by team (Consulta Externa, Gastro) and track personal KPIs (Hours, Shifts, Weekends).
- **Excel Import**: Seamlessly import roster data directly from Excel files.
- **Google Sheets Integration**: Import rosters from public Google Sheet links.
- **Responsive Design**: optimized for both desktop and mobile usage.
